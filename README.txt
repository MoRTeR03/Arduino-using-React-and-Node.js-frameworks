# Інструкція запуску проекту (ручний режим)

1. Відкрий проект у WebStorm у папці `robot-system/`.

2. Відкрий два термінали в WebStorm:

Термінал 1:
cd server
npm install
npm start

Термінал 2:
cd client
npm install
npm start

3. Прошивка ESP32 та Arduino залишається без змін, як в попередньому архіві.
